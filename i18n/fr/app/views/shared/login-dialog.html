Comment m’inscrire pour obtenir un compte CDB?

Toute personne peut avoir un compte CBD qui lui permettra d'enregistrer des informations dans toutes les centres d’échage d’information de la CDB, y compris le Centre d'échange pour la prévention des risques biotechnologiques, le Centre d'échange sur l'accès et le partage des avantages et le Centre d'échange en vertu de la Convention. Vous pouvez vous inscrire pour créer un compte au accounts.cbd.int/signin

Assure-vous d’entrer toutes les informations requises, tel qu’indiqué dans le formulaire. Vous recevrez automatique un courriel avec un hyperlien sur lequel vous devrez cliquer pour confirmer votre compte.

Vous avez oublié votre mot de passe ?

Cliquez ici accounts.cbd.int/password/reset pour un nouveau mot de passe. On vous demandera votre adresse électronique et, si vous êtes un utilisateur inscrit, un courriel vous sera immédiatement envoyé à votre compte de messagerie électronique inscrit avec les instructions à suivre pour créer un nouveau mot de passe.
