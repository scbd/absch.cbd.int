Renseignements sur l'organisme
Le nom scientifique d'un organisme est habituellement composé du nom du genre suivi de la dénomination spécifique. Par exemple : Le nom scientifique du riz est oryza sativa.
La classification taxonomique est la désignation formelle d'un organisme. Exemple : genre Oryza; espèce : sativa
Un nom commun est utilisé pour référer à un organisme dans la langue de tous les jours et peut comprendre des synonymes. Exemple: maïs / blé d'Inde.
